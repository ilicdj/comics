## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/ilicdj/comics.git
   ```
2. Navigate to the project directory:
   ```bash
   cd comics
   ```
3. Install dependencies:
   ```bash
   npm i
   ```
4. Start the project:
   ```bash
   npm start
   ```
